# 问题描述
表述一：m 个球每次有放回的、相互独立的取 n 次，被取最多次数的球期望被取的次数是多少。  
表述二：n个球放入m个箱子里面，球最多的箱子里面期望有多少个球？  
表述三：n个元素插入长度为m的哈希表，使用开链法解决冲突，最长的链表期望是多少？

看到有个结论是当 m=n 时，最多被取的次数期望时 O(logn/loglogn)
